## 🎤프로젝트 소개
- 개인 플레이리스트 파일을 만들고, 사용자가 직접 노래를 직접 설정하여 만드는 플레이리스트
- 유튜브 링크를 통해서 노래 감상가능
- 다른 사용자와 서로의 플레이리스트 공유 가능
